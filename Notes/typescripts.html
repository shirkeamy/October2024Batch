TypeScript Notes

TypeScript एक typed Javascript है| Typesript मे Javascript के syntax को ही type दिया जाता है, इसी के कारण हमे कोड रन करणे से पहले हि हमे Error मिल जाता है|

TypeScript एक Open-Source programming language है| जो सभी browsers पे रण होती है| जिस browser या Environment पे Javascript run होती है उस browser पे TypeScript भी run होती है| TypeScript को हम Javascript का superset भी बोल सकते है|

जब हम TypeScript का code लिखते है तब ओ code TypeScript compiler compile करके javascript मी convert करता है और यही code हम final मी run करते है|

TypeScript file का extention .ts होता है और javascript के file का .js होता है|
